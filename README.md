# YAMA Lokman

**YAMA Lokman** sağlık okuryazarlığı, anamnez oluşturma ve doktora hazırlık amacıyla tasarlanan web tabanlı bir bilgi yoldaşıdır.

## Temel ilke

> Teşhis koymaz; anlamaya yardım eder.

## Motto

> Bedenini tanı. Bilgini artır. Gerektiğinde hekime danış.

## Amaç

YAMA Lokman doktorun yerine geçmez, teşhis koymaz, tedavi veya ilaç tavsiyesi vermez.  
Temel amacı kullanıcının kendi bedenini, belirtilerini, tahlil değerlerini ve sağlık öyküsünü daha iyi anlamasına yardımcı olmaktır.

## İlk modüller

1. Sağlık Öyküm / Anamnez
2. Beden Atlası
3. Tahlil Değerlerini Anla
4. Belirtiyi Anla
5. Doktora Hazırlan
6. Acil Uyarı İşaretleri
7. Yanlış Bilinenler
8. Sağlık Notlarım

## İlk geliştirme hedefi

İlk sürüm Chrome, Safari ve mobil tarayıcılarda çalışan sade bir web demo olarak hazırlanacaktır.

## Güvenlik sınırı

Bu uygulama tıbbi teşhis, tedavi veya ilaç önerisi sunmaz.  
Acil veya ciddi belirtilerde kullanıcı hekime veya acil sağlık hizmetlerine başvurmalıdır.
